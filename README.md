
# Brainstorm-Democracy-V2

Group brainstorming - enter your ideas - then vote on them.

## how

### install

```
git clone https://github.com/mel-booth/Brainstorm-Democracy-V2
cd
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
